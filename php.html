Créer une page dynamique

Une page dynamique est une page qui réagit aux données fournies par l'utilisateur en modifiant le contenu de la page.

Organisez les pages en blocs fonctionnels

Pour structurer ses pages en blocs fonctionnels, on utilise des inclusions. On écrit un code qui a besoin d'être répété plusieurs fois dans une page PHP qu'on inclue dans chaque page.

Effectuer des requêtes MySQL

MySQL utilise des tables contenant des champs qui structurent les données de la table. On peut ainsi retrouver les données que l'on souhaite à l'aide d'une requête.

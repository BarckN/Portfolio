Les dictionnaires

Afin de mieux utiliser les données, les dictionnaires ont été créé. Ils enregistrent des données dans des paires clés-valeurs. Un exemple est "couleur_preferee": "Noir". Ainsi, pour accéder à la valeur noir, il faut spécifier : mon_dico['couleur_preferee'].

Création d'une fonction

Une fonction est un bloc de code avec un but spécifique, auquel vous pouvez donner un nom. Quand vous appelez cette fonction, vous exécutez le code qu’elle contient. Les fonctions vous laissent saisir des paramètres pour exécuter le même code sur différentes valeurs. Les fonctions aident à séparer le code en sections plus petites pour garder le fil sur ce que chaque partie est censée faire.

Les fonctions vont pouvoir être composés de dictionnaires, de boucles ou de conditions.

L'extraction de données web

Le package Requests permet d'obtenir le code HTML de la page web désirée. Il faut sauvegarder l’URL que nous voulons extraire dans une variable url , et utiliser la méthode .get() pour récupérer les données HTML.
